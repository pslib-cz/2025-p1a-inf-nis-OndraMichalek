# BanManager

BanManager je informační systém určený pro správu a evidenci banů na Minecraft serveru. 
Systém funguje jako plugin propojený se serverem, který zaznamenává bany udělené pomocí administračních pluginů (například Essentials) do databáze. 

Nasbíraná data jsou následně zobrazována na veřejném webovém rozhraní, kde si mohou uživatelé prohlížet informace o banech. Součástí systému je také možnost přihlášení pomocí Microsoft účtu, přes který hráč hraje na minecraft serveru, za účelem ověření identity hráče. Po přihlášení může uživatel vytvořit ticket nebo námitku ke svému banu a komunikovat s administrátory serveru.

Cílem projektu je vytvořit přehledný systém pro správu trestů a zjednodušit komunikaci mezi hráči a vedením serveru.
